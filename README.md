# Hexo Renderer Sass

[![npm-image]][npm-url]
[![lic-image]](LICENSE)

Sass renderer plugin for Hexo.

## Installation

![size-image]
[![dm-image]][npm-url]
[![dt-image]][npm-url]

```bash
$ npm install hexo-renderer-ruby-sass
```

## Options

You can configure this plugin in Hexo `_config.yml`. Default options:

```yaml
sass:
  debug: false
  outputStyle: nested
  sourceComments: false
  sourceMapEmbed: false
```
